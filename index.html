Jaasbook

All about the Java Authentication & Authorization Service - source code and other material for the unpublished JAAS in Action (circa 2005)

Back in 2005 I was asked to write a book on JAAS, the Java Authentication and Authorization Service. The book didn't end up getting published, but here's the result.

I haven't checked this code in a long time, but it consistently gets comments over on JaaSBook.com. Hopefully I'll get around to fixing it up. I'm happy to have some folks help, of course.
